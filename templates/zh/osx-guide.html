OS X dVPN 教程

安装OpenVPN

Tunnelblick是一个免费的开源OpenVPN客户端，适用于Mac OS X。您可以从Tunnelblick下载页面下载最新的磁盘映像。双击下载的.dmg文件并按照提示进行安装。



配置OpenVPN

安装过程快结束时，Tunnelblick将询问您是否有任何配置文件，选择否。下载Veles配置文件并双击Veles.ovpn，Tunnelblick将安装客户端配置文件（需要管理权限）。



连接OpenVPN

双击启动Tunnelblick。启动Tunnelblick后，屏幕右上角的菜单栏中将显示一个Tunnelblick图标，用于控制连接。单击图标，然后单击连接菜单项启动VPN连接。选择veles进行连接。

安装 Stunnel

按 Command+Space 输入 Terminal 然后按 enter/return 键. 在终端中运行下面的命令，然后按enter/return 键.

  • ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)" < /dev/null 2> /dev/null

如果提示您输入密码，请输入Mac的用户密码。当你输入密码时，密码不会显示在屏幕上。所以只要输入密码并按回车键就可以了。然后输入如下命令：

  • brew install stunnel


配置 Stunnel

下载 Stunnel 客户端配置, Stunnel 客户端证书 .

下载完成后，进入Stunnel目录/usr/local/etc/Stunnel/，复制下载的veles.Stunnel.conf和Stunnel.pem

可能需要将veles.stunnel.conf重命名为stunnel.conf


  • ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)" < /dev/null 2> /dev/null

下载Veles配置文件并双击veles-shielded.ovpn。Tunnelblick将安装客户端配置文件。需要管理权限。



用Stunnel连接OpenVPN客户端

按 Command+Space 输入 Terminal 然后按 enter/return 键. 我们通过如下命令开启Stunnel：

  • stunnel

现在双击应用程序文件夹中的Tunnelblick启动Tunnelblick。启动Tunnelblick后，屏幕右上角的菜单栏中将显示一个Tunnelblick图标，用于控制连接。单击图标，然后单击“连接”启动VPN连接。选择veles-shielded连接。

测试您的VPN连接

一旦所有的东西都安装好了，只需要一个简单的检查就可以确认他是否运作。在未启用VPN连接的情况下，打开浏览器并转到 DNSLeakTest。
该站点将返回由您的网络服务提供商分配的IP地址，以及您在世界其他地方显示的IP地址。若要通过同一网站检查您的DNS设置，请单击扩展测试（Extended Test），它会告诉您正在使用的DNS服务器。
现在，连接到您的VPN客户端并刷新浏览器，应该会显示与您的VPN服务器的完全不同的IP地址。然后再次进行扩展测试（Extended Test）检查您的DNS设置，并确认您正在使用Veles dVPN推送的DNS解析。
